# UI/UX Spec: Story 6.4 - Keyboard Shortcuts
**Shortcuts:**
- "/" - Open search
- "Esc" - Close modals
- "?" - Show help dialog
- "N" - New brief
- "G D" - Go to dashboard
**Help Dialog:** Shows all shortcuts, grouped by category
